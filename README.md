## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the required libraries.
```bash
pip install numpy 
pip install pandas
pip install sklearn
pip install math
pip install keras
pip install cv2
```
The list of libraries will be updated with the time.
